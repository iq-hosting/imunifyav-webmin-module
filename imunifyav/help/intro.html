ImunifyAV Webmin Module
This module provides seamless integration of the ImunifyAV(+) standalone UI into Webmin, allowing you to manage ImunifyAV directly from your Webmin control panel.

Key Features:

  • Secure UI Integration - The ImunifyAV interface is loaded via secure iframe with Content-Security-Policy protection.
  • IP Restriction - Access to the ImunifyAV UI is automatically restricted to your current session IP address only.
  • Token-Based Authentication - No credentials needed! The module uses ImunifyAV's token-based authentication which refreshes on every page load.
  • Notification System - Configure Telegram and Email notifications for scan events including malware detection.

Security Features:

  • Content-Security-Policy (CSP) headers prevent XSS attacks
  • Dynamic .htaccess updates ensure only the root user's IP can access the UI
  • Input validation and sanitization on all user inputs
  • Secure token handling with no shell command injection vulnerabilities
  • File locking to prevent race conditions
  • Action logging for audit trails

Requirements:

  • ImunifyAV(+) installed in standalone mode
  • Properly configured integration.conf file
  • Apache web server with mod_rewrite enabled
  • Valid hostname pointing to UI installation path
